Table of Contents
Este é um Balão de Nota. Servem para lhe fornecer informações adicionais sobre o tópico a ser tratado.
Este é um Balão de Dica. O seu objetivo é dar dicas pequenas mas práticas e úteis.
Este é um Balão de Importância. Significa que a informação contida é de grande importância, não devendo assim ser ignorada.
Este é um Balão de Cuidado. Quando encontrar um deve prestar atenção ao seu conteúdo e ter cuidado para não causar erros.
Este é um Balão de Aviso. A informação contida deve ser levada muito a sério e, caso seja ignorado, pode dar origem a problemas ou situações desagradáveis.
