Esta página mostra os requisitos de software e hardware de duas perspetivas diferentes:

  • Para os desenvolvedores Java que criam jogos; e

  • Para os utilizadores (os seus clientes) que jogam os seus jogos.

Para Desenvolvedores

Estes são os requisitos mínimos para desenvolvedores que cria jogos usando o jMonkeyEngine SDK:

Sistema OperativoMac OS X, Windows, Linux, Solaris

Memória (tamanho do heap do JVM)

> 40 MB ++ memória para os assets

CPU

> 1 GHz

Placa gráfica

AMD/ATI Radeon 9500, NVIDIA GeForce 5 FX, Intel GMA 4500, ou superior
suporte a OpenGL 2.0 ou superior (as livrarias nativas estão incluídas com o download)

Java Development Kit

JDK 7 ou maior
A sua equipa de desenvolvimento obtém o Java Development Kit gratuitamente daqui: http://www.java.com; para Mac OS, ver apple.com.
Um nível intermédio de experiência com Java é necessário

Recomendamos o uso do jMonkeyEngine SDK como a ferramenta de desenvolvimento do jogo (IDE). No entanto, qualquer outro IDE Java (e até um editor de texto e um terminal) pode ser usado e funciona bem com a framework jME, caso já tenha experiência em usar essa ferramenta. Para os requisitos de outros IDEs, consulte a documentação de terceiros.

Para Utilizadores

Estes são os requisitos mínimos para os seus clientes, os utilizadores que jogam jogos criados com o jMonkeyEngine3:

Sistema OperativoMac OS X, Windows, Linux, Solaris

Memória (tamanho do heap do JVM)

> 10 MB ++ memória paraos assets (modelos, texturas, sons, etc, dum jogo em particular)

CPU

> 1 GHz

Placa gráfica

AMD/ATI Radeon 9500, NVIDIA GeForce 5 FX, Intel GMA 4500, ou superior
suporte a OpenGL 2.0 ou superior (as livrarias nativas estão incluídas com o download)

Dispositivos Android

(Apenas para jogos móveis) Android 2.2 OS, placa gráfica que suporte OpenGL 2

Java Runtime

Java 5 ou maior
O Java Virtual Machine (JVM) é necessário para correr jogos jME.
Geralmente o JVM já vem pré-instalado, caso não tenham, os seus utilizadores podem obtê-lo aqui: http://www.java.com; para Mac OS, ver apple.com.

Certifique-se de que informa os seus utilizadores quanto aos requisitos. Make sure to inform your users about these requirements.
