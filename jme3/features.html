Kit de Desenvolvimento de Software: jMonkeyEngine SDK

  • Cria projetos java prontos para jME3

    • Classpath pré-configurado

    • Vem com um JDK compatível

    • Vem com ferramentas de conversão para Blender e mais

    • Asset Manager para carregar ficheiros multimédia e modelos 3D +incluíndo conclusão de código para assets

    • Scrips Ant não-proprietárias de build

    • Popups Javadoc para jME3, projetos padrão, e palete de peças de código

  • Editor Java e XML cheio de funcionalidades

  • Plugins

    • File Version Control

    • Debugger e Profiler (opcional)

    • Conversores e Importadores para assets de jogo (modelos 3D etc)

    • 3D Scene Viewer e Scene Composer

    • Editor de Materiais

    • Editor de Shader Nodes

    • Gerador de terreno, ferramentas de pintura e edição

    • Criador personalizado de fontes

    • Suporte para Asset Packs personalizados com modelos, texturas e mais

    • Criador de texturas Procedural (NeoTexture)

    • Gerado de Level of Detail (LOD)

    • …​ e muito mais…​

  • Distribuição

    • Cria executáveis desktop para Win, Mac, Linux

    • Cria executáveis mobile para Android, o suporte para iOS está em desenvolvimento.

    • Cria Applets JNLP WebStart e Java Browser

  • Baseado na plataforma NetBeans

    • Suporta todos os plugins do NetBeans IDE

Formatos Suportados

  • Modelos: Blender (.blend)

  • Modelos: Modelo Ogre3D (.mesh.xml, .skeleton.xml, .material), Ogre3D dotScene (.scene)

  • Modelos: Wavefront (.OBJ, .MTL)

  • Modelos: Collada

  • Modelos: 3DS

  • Texturas: .DDS, .HDR, .PFM, .TGA, .JPG, .PNG, .GIF.

  • Fontes: BMFont fonts (.FNT)

  • Áudio: Waveform (.WAV), Ogg/Vorbis (.OGG)

  • Ficheiros binários do jME3 (modelos e cenas): .j3o

  • Materiais do jME3: .j3m

  • Definições de materiais do jME3: .j3md

  • Filtros post-processing do jME3: .j3f
