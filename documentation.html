Também é muito bem-vindo a reparar quaisquer erros tanto de informação como de escrita, assim como parágrafos vagos, usando o menu “Wiki” no topo ou os botões de “Edit” ao pé de cada parágrafo. Tem de ter sessão iniciada para poder editar esta wiki.

Isto é um Balão. Irá encontrar muitos destes espalhados pela Wiki, Veja os Significados dos Balões para saber o significado deste e de todos os outros Balões.

* O SDK cria projetos baseados em Ant que qualquer IDE de Java pode importar. Recomendamos aos utilizadores doutras IDEs para tabém descarregarem o jMonkeyEngine SDK e escolherem File ▸ Import Project ▸ External Project Assets para criar um projeto sem código para gerir só os assets. Desta forma pode programar em qualquer IDE à sua escolha, e usar o SDK para converter os seus modelos para o formato .j3o.
